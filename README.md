This is a Meteor-react boilerplate using Meteor 1.4 and React 15. It has been put together as a baseline for learning Behavior/Test-Driven-Design.

### First run of the app

```bash
npm install
```
### Running the app

```bash
npm start
```

### Scripts

To lint:

```bash
npm run lint
```

Run tests with Chimp:

```bash
npm run chimp
```
